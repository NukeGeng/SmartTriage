Client Backend AI / ML Dữ liệu Chức năng cốt lõi

Luồng dữ liệu: Người dùng → Frontend (gọi API kèm JWT) → Backend. Từ Backend rẽ hai nhánh: sang AI Service (HTTP /analyze-ticket) và xuống PostgreSQL (SQL/ORM). Backend là nơi duy nhất truy cập cơ sở dữ liệu nghiệp vụ; AI Service chỉ đọc/ghi Model artifacts. Khi tạo phản ánh, AI thực hiện hai chức năng cốt lõi (phân loại + ưu tiên) rồi lưu vào ticket_analyses; phản ánh đã xử lý quay lại nuôi Training Pipeline — khép kín vòng phản hồi học máy.
